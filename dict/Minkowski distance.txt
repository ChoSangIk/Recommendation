Minkowski distance 

- The Minkowski distance is a metric in a normed vector space which can be considered as a generalization of both the Euclidean distance and the Manhattan distance.
(유클리디안 거리측정과 맨하탄 거리측정의 일반화라고 할 수 있는 놈 벡터 공간의 metric, 실제로 p=1일때, 맨하탄 거리측정이 되고, p=2일때, 유클리디안 거리측정이 된다.)

Metric

- In mathematics, a metric or distance function is a function that defines a distance between each pair of elements of a set.
(집합의 한 쌍의 요소 사이의 거리를 측정하는 함수)

Normed vector space

- In mathematics, a normed vector space is a vector space on which a norm is defined.
(norm이 정의된 벡터 스페이스)
